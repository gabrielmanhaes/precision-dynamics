# Methodology Statement

## AI-assisted synthesis

This framework was developed using AI-assisted literature synthesis.
The core theoretical hunch — that depression, PTSD, psychosis, and ADHD
are the same precision system miscalibrated in different directions —
preceded any literature review. The ODE implementation, empirical target
identification, mechanism fixes, and theoretical extensions were developed
through an iterative process combining AI-generated synthesis with
verification against primary literature.

**Implications for interpretation:**

1. Every mechanism identified in this framework should be independently
   verified against comprehensive literature searches. The AI system used
   for synthesis was trained on scientific literature that may include the
   mechanisms identified — it is not possible to fully distinguish genuine
   novel derivation from sophisticated pattern completion over the training
   corpus.

2. The three mechanism fixes (psilocybin cortisol direction, REM alpha
   magnitude, NREM LZW ordering) were each identified by the synthesis
   process and then found in specific published papers not used as
   training targets. These represent cases where the process pointed to
   real published mechanisms. Whether this constitutes discovery or
   retrieval is genuinely unclear.

3. The bidirectional HRV treatment prediction is the cleanest test of
   whether this framework contains genuine novel insight: it is a specific
   quantitative prediction about existing clinical datasets that has not
   been reported in the literature. If confirmed, the methodology produced
   a valid novel prediction regardless of how it was generated.

## Reproducibility

The synthesis process is reproducible in principle: the same AI-assisted
approach, applied to the same literature, by a different researcher,
should arrive at a similar framework. This reproducibility is itself a
form of validation — the framework is not idiosyncratic to one person's
intuitions but follows from the structure of the existing literature when
synthesized across domains.

## No new empirical data

This framework is entirely computational and theoretical. No human
subjects were involved. No new experiments were conducted. All empirical
targets are from published literature as cited. The model makes
predictions about existing and future data; it does not itself constitute
empirical evidence.

## Independent validation status

As of the current version:
- 1 genuinely prospective prediction confirmed: psilocybin heart rate
  direction (Holze et al. 2024, published after model finalization)
- 7 post-hoc comparisons with correct directional alignment
- 1 failed prediction: antidepressant delay acceleration (contradicted
  by STAR*D data, prediction narrowed)
- 0 prospective clinical predictions yet tested in datasets

The framework requires prospective empirical validation before any
clinical application. OSF preregistration of the bidirectional HRV
prediction is recommended before any dataset analysis.
