# Falsifiability Criteria

This document specifies the conditions under which each major claim
of the precision dynamics framework would be disconfirmed.

---

## Claim 1: P_selfmodel constitutes conscious experience amplitude

**Predicted:** P_selfmodel reduction = consciousness amplitude reduction.
Ego dissolution = P_selfmodel approaching zero.

**Key falsifiability test — propofol dissociation:**

Propofol anesthesia and high-dose psilocybin both produce:
- Similar DMN coherence reduction (~0.45 normalized)
- Loss of unified conscious experience
- Strong reduction in self-report measures

They should differ in:
- Post-acute plasticity window: psilocybin produces therapeutic TEMP
  (heterosynaptic plasticity, DMN reorganization, lasting personality
  change). Propofol produces none of these.
- Mechanism: psilocybin = 5-HT2A agonism -> increased cortical entropy ->
  TEMP -> downstream plasticity. Propofol = GABA-A potentiation ->
  thalamocortical suppression -> no TEMP.

**Disconfirmation criteria:**
- If propofol produces equivalent post-acute plasticity and therapeutic
  outcomes to psilocybin at matched DMN coherence levels -> the
  P_selfmodel mechanism is wrong (mechanism is something else)
- If psilocybin-induced ego dissolution does NOT correlate with DMN
  coherence reduction across subjects -> DMN is wrong proxy for P_selfmodel
- If DMN coherence does NOT decrease monotonically with psychedelic dose
  across published datasets -> ordering prediction disconfirmed

**What would survive disconfirmation:**
Even if P_selfmodel != consciousness amplitude, the ODE model, HRV
biomarker, and bidirectional treatment prediction are independent of
this claim and would remain valid.

## Claim 2: Bidirectional HRV treatment selection

**Predicted:** High-HRV depression -> better psilocybin response.
Low-HRV depression -> better SSRI response.

**Preregistration requirement:** This prediction must be preregistered
with specific numerical criteria before dataset analysis. Suggested
criteria: >=10% difference in remission rate between high-HRV and
low-HRV groups within each treatment arm, in opposite directions
across treatment arms.

**Disconfirmation criteria:**
- If high-HRV depression shows equal or worse psilocybin response than
  low-HRV depression -> directional prediction wrong
- If HRV does not correlate with any treatment outcome measure -> HRV
  is not a precision proxy
- If the same HRV cutoff predicts response in the same direction for
  both psilocybin and SSRIs -> not bidirectional, possibly a general
  severity marker

**Datasets where this is testable without new data collection:**
- COMPASS psilocybin trial (Goodwin et al. 2022, NEJM, N=233):
  contact corresponding author for HRV data availability
- STAR*D (Rush et al. 2006, citalopram, N~4000): NIMH publicly
  available; check for ECG/HRV measures in baseline assessment
- Imperial College psilocybin vs escitalopram trial
  (Carhart-Harris et al. 2021, NEJM, N=59): contact for HRV data

## Claim 3: REM sleep as endogenous TEMP

**Predicted:** REM disruption -> reduced plasticity maintenance ->
progressive canalization -> treatment resistance. Restoration of REM ->
improved treatment response.

**Disconfirmation criteria:**
- If REM suppression (e.g., via REM-suppressing antidepressants like
  MAOIs) does not reduce long-term treatment efficacy despite acute
  response -> REM plasticity maintenance hypothesis wrong
- If SD(LZW) does not differ between depressed patients and healthy
  controls -> complexity variability prediction disconfirmed
- If successful treatment does not restore SD(LZW) toward healthy
  baseline -> treatment mechanism prediction disconfirmed

## Claim 4: Dream-meditation-psychedelic computational equivalence

**Predicted:** Individual differences in P_selfmodel reduction capacity
should predict performance across all these domains simultaneously.
One latent factor should underlie:
- Meditation depth (attentional stability in low-P states)
- Lucid dreaming frequency (P_selfmodel reduction during REM with
  metacognition preserved)
- Psychedelic response depth (mystical experience score)
- Hypnagogic creativity (cross-domain insight at sleep onset)
- Absorption (Tellegen Absorption Scale)

**Falsifiability test:** Factor analysis of a survey measuring all 5
domains in N>=200 participants. If they do NOT load on a single latent
factor, computational equivalence is disconfirmed.

**Disconfirmation criteria:**
- If absorption, meditation depth, lucid dreaming, and psychedelic
  response load on 3+ independent factors with no dominant shared
  factor -> the unification claim is wrong
- If a person who is an excellent meditator shows systematically poor
  psychedelic response (or vice versa) across N>=50 such cases ->
  these are different capacities with different underlying mechanisms
